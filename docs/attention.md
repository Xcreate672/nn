# 注意力机制 (Attention)

## 概述

注意力机制（Attention Mechanism）是一种让模型在处理信息时，
能够动态地关注输入中最相关部分的技术，广泛应用于自然语言处理和计算机视觉领域。

## 主要特点

- **动态权重**：根据输入内容动态分配注意力权重
- **全局依赖**：能够捕捉序列中任意位置之间的依赖关系
- **可解释性**：注意力权重可以直观展示模型关注的区域

## 计算过程

### 1. 查询、键、值

- Query (Q)：查询向量，表示当前关注的内容
- Key (K)：键向量，表示每个位置的标识
- Value (V)：值向量，表示每个位置的实际信息

### 2. 注意力分数

计算 Q 和 K 的点积相似度，再除以缩放因子后用 softmax 归一化：

`Attention(Q, K, V) = softmax(Q * K^T / √d_k) * V`

其中 d_k 是键向量的维度，用于防止点积过大导致梯度消失。

### 3. 加权求和

用注意力分数对 V 加权求和，得到最终输出向量。

## 主要变体

### 自注意力（Self-Attention）

- Q、K、V 来自同一个输入序列
- 用于捕捉序列内部各位置之间的关系

### 多头注意力（Multi-Head Attention）

- 并行运行多组注意力机制
- 能从不同角度捕捉输入的不同特征

## 与自动驾驶的结合

- **场景理解**：对摄像头图像中的不同区域分配注意力权重，聚焦于行人、车辆等关键目标
- **轨迹预测**：对历史轨迹序列的关键帧赋予更高权重
- **多传感器融合**：对雷达、摄像头等不同传感器的输出动态加权融合
