# 循环神经网络

## 概述

循环神经网络（Recurrent Neural Network, RNN）是一种专门处理序列数据的深度学习架构，
通过在网络中引入循环连接，使模型能够记忆历史信息，适用于时间序列、自然语言等任务。

## 主要特点

- **序列建模**：能够处理任意长度的序列输入
- **参数共享**：在序列的每个时间步共享相同的权重
- **记忆能力**：隐藏状态可以捕捉历史序列信息

## 基本结构

### 1. 输入层 (Input Layer)

- 接收序列中每个时间步的输入向量
- 输入维度由任务特征数量决定

### 2. 隐藏层 (Hidden Layer)

- 包含循环连接，将上一时间步的隐藏状态传递到当前时间步
- 隐藏状态公式：`h_t = tanh(W_h * h_{t-1} + W_x * x_t + b)`

### 3. 输出层 (Output Layer)

- 根据任务类型输出预测结果
- 可以是每个时间步输出，也可以是最终时间步输出

## 主要变体

### LSTM（长短期记忆网络）

- 引入遗忘门、输入门、输出门解决长期依赖问题
- 能有效捕捉长距离序列依赖关系

### GRU（门控循环单元）

- LSTM的简化版本，使用更新门和重置门
- 参数更少，训练速度更快

## 常见问题

- **梯度消失**：标准RNN难以学习长距离依赖，通常用LSTM/GRU解决
- **梯度爆炸**：通过梯度裁剪（Gradient Clipping）缓解

## 应用领域

- 自然语言处理（文本分类、机器翻译）
- 语音识别
- 时间序列预测
- 自动驾驶中的轨迹预测
