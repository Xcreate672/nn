# CARLA IMU 数据采集平台开发汇报文档
## 一、项目概述
本项目是基于CARLA自动驾驶模拟器的**IMU惯性测量单元数据采集与可视化驾驶平台**，旨在为驾驶员行为识别研究提供标准化的数据采集环境。平台支持手动/自动驾驶双模式，具备完整的车辆控制、状态显示、数据自动保存功能，操作友好且数据格式规范。

## 三、问题修复记录
### 1. 初始运行报错修复
- **错误1**：`AttributeError: 'Namespace' object has no attribute 'rolename'`
  - 原因：代码依赖`--rolename`命令行参数但未设置默认值
  - 解决方案：在World类初始化中设置默认角色名"hero"，同时在参数解析中添加默认值

- **错误2**：`Boost.Python.ArgumentError: spawn_actor参数不匹配`
  - 原因：CARLA API要求传入`carla.Transform`对象，错误直接传入`carla.Location`
  - 解决方案：所有传感器生成统一使用`carla.Transform(carla.Location(...))`格式

- **错误3**：`SyntaxError: name 'world' is assigned to before global declaration`
  - 原因：`global world`声明位置错误，出现在变量赋值之后
  - 解决方案：删除不必要的全局变量声明，改为通过函数参数传递world对象

- **错误4**：`NameError: name 'world' is not defined`
  - 原因：HUD渲染函数直接访问全局world变量，未通过参数接收
  - 解决方案：修改`HUD.render()`方法签名，增加world参数，由World类调用时传入自身实例

## 四、最终完整功能说明
### 1. 基础运行功能
- ✅ 自动连接CARLA模拟器，生成随机车辆
- ✅ 实时采集IMU传感器数据（加速度X/Y/Z、陀螺仪X/Y/Z）
- ✅ 程序退出时自动将数据保存为CSV文件，命名格式`out_{用户名}.csv`
- ✅ 支持自定义服务器地址、端口、车辆类型和用户名

### 2. 界面交互功能
- ✅ 中英文双语界面，按空格键一键切换
- ✅ 左侧信息面板实时显示：服务器/客户端帧率、车辆型号、地图名称、行驶速度、驾驶模式、车辆状态
- ✅ 底部提示区域显示操作反馈信息
- ✅ 鼠标拖拽屏幕空白区域可360°旋转视角

### 3. 驾驶控制功能
#### 自动驾驶模式
- 车辆自动沿道路行驶
- 点击"切换模式"按钮可切换至手动驾驶

#### 手动驾驶模式（圆形轮盘操控）
- **圆盘轮盘**：按住白色圆盘拖动控制车辆
  - 向上拖动：前进加速（0.2-0.8倍油门）
  - 向下拖动：倒车后退（0.2-0.7倍油门）
  - 左右拖动：控制转向（最大0.8倍转向角）
  - 轮盘归中：自动刹车
  - 松手自动复位至中心
- **独立手刹按钮**：点击切换手刹开关，开启时按钮变为红色高亮
- **模式切换**：随时切换回自动驾驶模式

### 4. 辅助功能
- ✅ 一键打开输出文件夹按钮，直接定位到CSV文件保存目录
- ✅ ESC键一键退出程序，自动清理CARLA中的车辆和传感器
- ✅ 车辆状态实时识别与显示：直行、加速、刹车、左转、右转、倒车、手刹

## 五、运行指南
### 1. 环境准备
- 安装CARLA模拟器（推荐0.9.14版本）
- 安装Python 3.7（必须版本）
- 安装依赖库：
  ```bash
  pip install pygame numpy pandas carla
  ```

### 2. 文件放置
将代码文件`data_reciever.py`放入CARLA安装目录下的`PythonAPI/examples/`文件夹中。

### 3. 运行步骤
1. 启动CARLA模拟器：双击运行`CarlaUE4.exe`
2. 打开命令行，进入项目目录：
   ```bash
   cd D:\CARLA_0.9.14\PythonAPI\examples
   ```
3. 运行程序：
   ```bash
   py -3.7 data_reciever.py --name [你的用户名]
   ```
