# 支持向量机 (SVM)

## 概述

支持向量机（Support Vector Machine, SVM）是一种监督学习算法，
通过寻找最优超平面将不同类别的数据分隔开，广泛用于分类和回归任务。

## 主要特点

- **最大间隔**：寻找使两类数据间隔最大的分类超平面
- **支持向量**：只有少数关键样本（支持向量）决定超平面位置
- **核技巧**：通过核函数处理线性不可分的数据

## 基本概念

### 1. 超平面 (Hyperplane)

- 在n维空间中将数据分为两类的决策边界
- 二维中为直线，三维中为平面

### 2. 间隔 (Margin)

- 两类数据距超平面的最小距离之和
- SVM目标是最大化间隔

### 3. 支持向量 (Support Vectors)

- 距超平面最近的样本点
- 决定超平面的位置和方向

## 核函数类型

- **线性核**：适用于线性可分数据
- **RBF核（高斯核）**：适用于非线性数据，最常用
- **多项式核**：适用于特征交叉场景

## 与神经网络的对比

| 特性 | SVM | 神经网络 |
|------|-----|---------|
| 小样本 | 表现好 | 容易过拟合 |
| 大数据 | 速度慢 | 表现更好 |
| 可解释性 | 较强 | 较弱 |

## 应用领域

- 图像分类
- 文本分类
- 生物信息学
- 自动驾驶中的障碍物分类
